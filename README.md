# 📖 About

This repository contains multiple Streamlit applications created while learning the Streamlit framework.

The goal of this project is to understand how Python can be used to develop beautiful and interactive web applications with minimal effort.

The project covers:

* 📊 Data visualization
* 🎛️ Interactive widgets
* 📂 CSV file handling
* 📈 Dynamic charts
* 🤖 Machine Learning using Scikit-learn
* 🌸 Iris Flower Species Prediction

Whether you're new to Streamlit or exploring ML deployment, this repository serves as a great starting point.

# ✨ Features

### 🏠 Basic Streamlit Application

# 🚀 Installation

### Clone the repository

```bash
git clone https://github.com/Kirisaki00/Streamlit.git
```

### Navigate into the project

```bash
cd Streamlit
```

### Install dependencies

```bash
pip install streamlit pandas numpy matplotlib scikit-learn
```

---

# ▶️ Running the Applications

## Basic Streamlit App

```bash
streamlit run app.py
```

---

## Widgets Demo

```bash
streamlit run widgets.py
```

---

## Iris Flower Classifier

```bash
streamlit run classification.py
```
